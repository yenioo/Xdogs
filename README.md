## 📌 프로젝트 소개
애견인을 포함한 반려동물을 사랑하는 사람들의 커뮤니티 공간 Xdogs 입니다.
주요한 필수기능만 작동하고 있습니다. 참고 부탁드립니다.

- **프로젝트 일정**<br>
  2023.02.25 ~ 2023.03.01 (5일)
  
- **주요기능**
  - 로그인 및 회원가입
  - 정보수정
  - 게시판(게시판/게시물/댓글 추가,수정,삭제)
